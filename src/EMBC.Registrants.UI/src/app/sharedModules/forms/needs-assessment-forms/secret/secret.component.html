Security Phrase

The security phrase is a vital piece of information to ensure an Emergency Support Services (ESS) Responder can access your ESS File.

If you have been evacuated, please share the answer to this question with any household members that you are registering.

This will enable an Emergency Support Services (ESS) Responder to assist them when you are not able to be present.


Enter a memorable security phrase that is over 6 characters in length. You may include numbers and letters. The security phrase is not case sensitive.

For security reasons, your security phrase should not include your email address.
