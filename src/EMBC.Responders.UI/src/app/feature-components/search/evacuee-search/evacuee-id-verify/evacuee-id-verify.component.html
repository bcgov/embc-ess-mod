Evacuee Search & Registration

Can you present any government-issued photo ID to verify your identity?

Did the evacuee present any government-issued photo ID when the paper ESS File was completed?

Yes - the evacuee did provide valid government issued photo identification. No - the evacuee did not provide valid government issued photo identification. A selected option is required


Tips: Why ID?
+ -

Emergency Responders ask for ID so that they are able to access your Profile or Evacuation file, as well as issue you and your family with supports in the form of e-Transfer should you be eligible.

Unable to present ID?

If you are unable to present ID, you will still be provided with supports should you are eligible.

If you have a profile or evacuation file, you will be asked to confirm your security questions or security phrase to allow Emergency Responders to access your profile.


Tips: Accepted ID
+ -

ID must include a photograph of the evacuee as well as their full name and date of birth.

List of accepted government-issued ID:

  • Indigenous status card
  • Passport (an international passport is acceptable if it includes the name, date of birth, photo and signature of the applicant and is accompanied by a professionally translated version if not in French or English)
  • Driver's license
  • Enhanced Driver's License
  • Canadian military ID card
  • Government-issued ID card
  • Government-issued enhanced ID card
  • Health card Canadian citizenship card (issued before February 1, 2012)
  • Canadian permanent resident card
  • U.S. permanent resident card (green card)
